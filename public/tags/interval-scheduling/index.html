LeetCode 435: Non-overlapping Intervals

Today, let’s look at LeetCode problem 435: Non-overlapping intervals. The instructions are as follows: Given an array of intervals intervals where intervals[i] = [start_i, end_i], return the minimum number of intervals you need to remove to make the rest of the intervals non-overlapping. Note that intervals which only touch at a point are non-overlapping. For example, [1, 2] and [2, 3] are non-overlapping. Let’s dive in! Reformulation to a maximum-meetings problem This problem is very similar to another problem: counting the maximum number of meetings in one room. Put simply, each meeting has a start and an end time. The problem is to find how many meetings can be held in a single room without interfering with one another. The meeting times are given similarly as [start_i, end_i].
